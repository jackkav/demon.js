# demon.js

[click to see demo](http://demonic.surge.sh/)

## Start API
```bash
export DATABASE="mongodb://whatever"
cd persist && npm start
```

## Run Scraper
```bash
export API_LOCATION="http://myapi.com/api/"
cd scraper && npm start
```

## Run WebUI
```bash
export API_LOCATION="http://myapi.com/api/"
cd web && npm start
```
